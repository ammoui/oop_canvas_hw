# canvas_oop_hw

Учебный проект — векторный графический редактор, написанный в рамках курса по ООП и компьютерной графике.
Десктопное приложение построено на базе React и Tauri.

## Особенности реализации

- **Собственная растеризация**: фигуры рисуются попиксельно без использования высокоуровневых API Canvas.
- **Алгоритмы отрисовки**: реализованы алгоритмы Брезенхэма и Ву (со сглаживанием) для рисования линий.
- **Матричные преобразования**: перемещение, масштабирование и поворот фигур реализованы через умножение матриц 3x3 (`mat3`).
- **База фигур**: поддерживаются прямоугольники, линии, эллипсы, треугольники, а также кривые Безье (квадратичная, кубическая, составной путь).
- **Сохранение проектов**: реализована работа с локальной файловой системой через Tauri API (сохранение и загрузка сцен в формате JSON).
- **Управление слоями**: возможность изменять порядок отрисовки фигур.

## Стек технологий

- React + TypeScript
- Tauri (Rust)
- Vite
- Tailwind CSS

## Как запустить

Для запуска требуются Node.js и Rust.

Установка зависимостей:
```bash
npm install
```

Запуск десктопного приложения в режиме разработки:
```bash
npm run tauri dev
```

Запуск только веб-версии в браузере:
```bash
npm run dev
```

## Структура исходного кода

- `src/lib/math/` — реализация математики и матричных вычислений.
- `src/lib/raster/` — растеризатор (алгоритмы отрисовки линий и полигонов).
- `src/lib/shapes/` — классы фигур с поддержкой сериализации.
- `src/screens/` — React-компоненты экранов (Галерея, Редактор).
- `src-tauri/` — Rust-часть проекта (настройки окна и доступы к файловой системе).
